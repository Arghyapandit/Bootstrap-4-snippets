This is Bootstrap 4 latest version(4.4.x) `yrs 2020`.

## How to use

### Installation:

- Go to the vs code extension and click on install button, then restart the vs code.

### Bootstrap 4:

- To use the bootstrap 4 snippets in AngularJs, Angular 2+ or in normal HTML file, just type `b4` followed by pressing `Ctrl+Space`. List will be appears. Check the belows shortcuts.

### Bootstrap 4 shortcuts:
